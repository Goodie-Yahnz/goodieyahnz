Animating in Anim A Kri

Anim A Kri

is a very different animation app, released on 26th December 2024, by Goodie Yahnz


Unlike other animation apps;

 It uses CSS codes

 Its framerate is not 24 frames per second, rather 12.5 fps

 Its animation and drawing is done outside the app not inside

 Its frames are limited to 500 frames


How to use it is to create the drawings in a separate painting app, and save them. Save many more, creating each painting in the app, you will paste each of the images in the "code created frame" between the quotations i.e "0.0 {background-image: url('');}".

THINGS TO KNOW WHEN USING ANIM A KRI

EMBRACING COPY AND PASTE: Copy and Paste is a rule breaking animation rule that you have to follow. This is because the animation is outside and no onion skinning

SIMPLE DESIGNATION: For Anim A Kri is best to draw something really simple than to draw a very complex drawing. That's why it's recommended to use basic shapes rather than what is painted. This is to help easy animation. So the recommended apps for character designation are Microsoft Power Point, Paint and Keynote
